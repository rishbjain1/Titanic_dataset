# 🚢 Titanic - Machine Learning from Disaster

Predict which passengers survived the Titanic shipwreck using supervised machine learning.

> 🏆 **Achieved Top 10 on Kaggle's Titanic Leaderboard**  
> through optimized feature engineering and model tuning.

---

## 📁 Dataset

🔗 [Kaggle Competition Link](https://www.kaggle.com/competitions/titanic)

- `train.csv` – Includes features + survival outcome
- `test.csv` – Features only; survival is to be predicted
- `gender_submission.csv` – Sample submission format

Key columns: `Pclass`, `Sex`, `Age`, `Fare`, `Embarked`, `SibSp`, `Parch`, `Cabin`, etc.

---

## 📌 Tasks Performed

- ✅ Data Cleaning & Missing Value Imputation  
- ✅ Exploratory Data Analysis (EDA) & Visualization  
- ✅ Feature Engineering (e.g., Title extraction, Family size, Fare binning)  
- ✅ Model Building: Logistic Regression, Decision Trees, Random Forest, and more  
- ✅ Hyperparameter Tuning & Cross-Validation  
- ✅ Final Submission to Kaggle Leaderboard
